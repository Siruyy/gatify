# Contributing

The canonical contribution policy lives in:

- [`../CONTRIBUTING.md`](../CONTRIBUTING.md)

## TL;DR workflow

1. Branch from `dev`
2. Implement changes + tests
3. Run `make check`
4. Open PR to `dev`
5. Link the GitHub issue (`Closes #123`)

PRs should follow the repository PR template:

- [`../.github/PULL_REQUEST_TEMPLATE.md`](../.github/PULL_REQUEST_TEMPLATE.md)
