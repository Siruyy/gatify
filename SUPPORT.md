# Support

## Where to ask for help

Use the right channel so maintainers and contributors can help efficiently.

### Bug reports

Open a GitHub issue:

- [GitHub Issues](https://github.com/Siruyy/gatify/issues)

Please include:

- What you expected
- What happened
- Steps to reproduce
- Relevant logs/errors
- Environment details (OS, Go version, Docker version)

### Feature requests

Open a GitHub issue and label it as an enhancement request in the description.

### Usage questions and troubleshooting

Use GitHub Discussions:

- [GitHub Discussions](https://github.com/Siruyy/gatify/discussions)

### Security issues

For vulnerabilities, follow [SECURITY.md](SECURITY.md) and report privately.

## Response expectations

As an open-source project, response times can vary.

Maintainers will try to:

- Acknowledge new issues within a few days
- Request missing reproduction details when needed
- Prioritize bugs affecting stability and security
